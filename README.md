# React Learning Project

This project was bootstrapped with **Create React App**.

It is a **training project for learning React**, created while studying a React course by **Ivan Petrychenko**.  
The project is used for practicing React basics, components, props, state, and project structure.

---

## Getting Started

In the project directory, you can run:

### `npm start`

Runs the app in development mode.  
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload automatically when you make changes.  
Lint errors (if any) will be shown in the console.

---

## Purpose of the Project

- Learn React fundamentals
- Practice component-based architecture
- Understand project structure in Create React App
- Follow practical examples from the course

This project is **educational only** and not intended for production use.
